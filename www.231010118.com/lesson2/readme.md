		PHP常量
			常量 存储容器		里面的数据一旦定义就不能修改
			语法：define("常量名称","常量值");
			通常常量全部大写，多个单词中间用下划线
			
			当在输出的字符串中有变量时，将变量加上{ }
				echo "这位是{$name}同学";
